Welcome to Sustainable Challenges


  • The Game


    Completing any task will reward you with points that can be used to redeem vouchers or enter sweepstakes. There are also challenges that reward a higher number of points, and progress you to the next level.

  • Litter


    You can report instances of litter you see around campus by taking a photo of them and uploading them. Your report will then be submitted for approval. A game keeper will review it and either accept it or reject it. In the event it is acceptted, you will recieve points and progress your challenges.
    {% if is_staff %} As a game keeper, you can also report instances of litter however, you cannot approve your own report. There is also a heatmap to display litter hotspots around campus. {% endif %}

  • Events


    You can view upcoming and currently occurring events on campus as well as information related to them. Once you are at the location of the event, you can confirm that you have attended the event and receive points?. The confirmation button will not appear until the event is occuring.
    {% if is_staff %} As a game keeper, you can create events, and the location will be set to your current location/router access point. You can view the event details regardless of where you are accessing it from, and you can also renew the IP for the location in the scenario that the router IP is renewed. {% endif %}

  • Places of Interest


    As you walk around campus, you can view places of interest on the map and visit them. You must be close to the place to be able to say you've visited it, you can only recieve points for visiting a place once. The places of interest will have the theme of promoting sustainability.
    {% if is_staff %} As a game keeper, you can submit new places of interest and also visit them. {% endif %}

  • Forum


    You can create posts and comment on them. Posts can be endoresed by others and allow for discussions between users. These posts can also be used as a way to make recommendations on how the university can achieve its sustainability goals.
    {% if is_staff %} As a game keeper, you can moderate posts and remove them if necessary. You can create announcements which are placed at the top of the forum and are meant to be used as way to inform users of initiatives or changes. {% endif %}

  • Raffle


    There is a daily and weekly raffle you can enter for a chance to win more points. The daily raffle can be entered for free and the weekly raffle has an entry cost of 100 points. The raffle is drawn at 3pm daily, and at the same time on Friday for the weekly one.
